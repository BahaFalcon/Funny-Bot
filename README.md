```Funny Bot```
==========
### Что делает:
* #### Парсит изображения и анекдоты с сайта [**"pixmafia.ru"**](https://pixmafia.ru/)
  (анекдоты в список, изображения в файл)
* #### Создаёт базу данных и сохраняет полученные данные
* #### После запуска бота пользователь нажимает кнопку `анекдот` или `мемчик`, и получает рандомные тексты или изображения

### Установка
1. #### Создать бота в "BotFather", получить токен
2. #### Склонировать командой `git clone`
3. #### Установить зависимости `pip install -r requirements.txt`
4. #### Добавить полученный от "BotFather" токен в `main.py`
5. #### Запуск файлов `python3 launch.py`
6. #### Запуск бота `python3 main.py`
